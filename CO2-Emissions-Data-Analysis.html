I. Data Overview and Cleaning

The data set is from the official open data web site of the Canadian government. It depicts numerous details of how CO2 emissions for vehicles can differ based on certain characteristics. The data was collected over a seven year period, and there are 7,385 vehicles in this data set. Different vehicles with all of the same characteristics can appear more than once in the data set.

  1. Statistical Modeling

Four ANOVA models are built for this analysis. CO2 emissions is the dependent variable in all four of the models. The models are evaluated in order to see which one performs the best.

First, a one-way ANOVA is performed. The model displays CO2 Emissions as a function of the number of cylinders in a vehicle’s engine. The p-value of the variable representing the number of cylinders in a vehicle’s engine is extremely low (p < 0.001), thus it appears that the number of cylinders in a vehicle’s engine has a real impact on the CO2 emissions of the vehicle. The R-squared value for this model is 70%.

Second, another one-way ANOVA is performed. The model displays CO2 Emissions as a function of the fuel type of a vehicle. The p-value of the variable representing the fuel type of a vehicle is extremely low (p < 0.001), thus it appears that the the fuel type of a vehicle has a real impact on its CO2 emissions. The R-squared value for this model is 7.5%. When comparing both of the one-way ANOVA models, it is clear that cylinders has more explanatory power than fuel type by a large margin.

Third, a two-way ANOVA is performed. The model displays CO2 Emissions as a function of both the number of cylinders in a vehicle’s engine and the fuel type of a vehicle. Having both of these variables in the model improves the model’s performance because the residual variance of this model is lower than the previous two models. Also, both cylinders and fuel type are statistically significant (p-values < 0.001).

Fourth, another two-way ANOVA is performed. The model displays CO2 Emission as a function of the interaction between the number of cylinders in a vehicle’s engine and the fuel type of a vehicle. This is the best performing model because it has the smallest AIC of all the models.
